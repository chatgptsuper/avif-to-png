AVIF (AV1 Image File Format) é um formato de imagem moderno que oferece redução significativa de tamanho de arquivo em comparação com JPEG, PNG e WebP, mantendo alta qualidade visual. É baseado no codec de vídeo AV1.

Embora o AVIF ofereça melhor compressão, o PNG é mais amplamente suportado em aplicativos e plataformas. Converter para PNG garante compatibilidade com softwares que ainda não suportam o formato AVIF mais recente.
