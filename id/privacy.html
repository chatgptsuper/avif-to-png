Kebijakan Privasi

Terakhir diperbarui: 1 Januari 2025

Pendahuluan

Selamat datang di AVIFtoPNG ("kami," "milik kami," atau "kami"). Kami menghormati privasi Anda dan berkomitmen untuk melindungi data pribadi Anda. Kebijakan privasi ini akan memberi tahu Anda tentang bagaimana kami menangani data Anda saat Anda mengunjungi situs web kami dan memberi tahu Anda tentang hak privasi Anda.

Data yang Kami Proses

Layanan kami dirancang dengan memperhatikan privasi. Ketika Anda menggunakan konverter AVIF ke PNG kami:

  • Semua pemrosesan gambar terjadi secara lokal di browser Anda
  • Gambar Anda tidak pernah diunggah ke server kami
  • Kami tidak menyimpan atau memiliki akses ke gambar yang dikonversi
  • Tidak diperlukan akun pengguna atau pendaftaran

Data Penggunaan

Kami mengumpulkan data penggunaan anonim untuk membantu kami meningkatkan layanan kami. Ini mungkin termasuk:

  • Jenis dan versi browser
  • Sistem operasi
  • Situs web rujukan
  • Waktu dan tanggal kunjungan Anda
  • Halaman yang dilihat
  • Statistik penggunaan anonim (misalnya, jumlah konversi)

Data ini dikumpulkan menggunakan cookie dan teknologi serupa. Ini tidak termasuk informasi yang dapat diidentifikasi secara pribadi.

Cookie

Kami menggunakan cookie penting untuk memastikan fungsi yang tepat dari situs web kami. Kami juga menggunakan cookie analitik untuk memahami bagaimana pengunjung berinteraksi dengan situs web kami. Anda dapat mengontrol cookie melalui pengaturan browser Anda.

Layanan Pihak Ketiga

Kami menggunakan layanan pihak ketiga berikut:

  • Google Analytics: Untuk analisis penggunaan situs web
  • Layanan CDN: Untuk mengirimkan konten statis secara efisien

Layanan ini dapat mengumpulkan data sebagaimana dijelaskan dalam kebijakan privasi masing-masing.

Keamanan Data

Kami telah menerapkan langkah-langkah keamanan yang tepat untuk mencegah data Anda hilang secara tidak sengaja, digunakan, atau diakses dengan cara yang tidak sah. Karena gambar Anda diproses secara lokal di browser Anda, mereka tidak pernah terpapar pada risiko keamanan eksternal di pihak kami.

Hak Anda

Tergantung pada lokasi Anda, Anda mungkin memiliki hak tertentu mengenai data pribadi Anda, termasuk:

  • Hak untuk mengakses
  • Hak untuk perbaikan
  • Hak untuk penghapusan
  • Hak untuk membatasi pemrosesan
  • Hak untuk portabilitas data
  • Hak untuk menolak

Untuk menggunakan hak-hak ini, silakan hubungi kami menggunakan detail yang disediakan di bawah ini.

Perubahan pada Kebijakan Privasi Ini

Kami dapat memperbarui kebijakan privasi kami dari waktu ke waktu. Kami akan memberi tahu Anda tentang perubahan apa pun dengan memposting kebijakan privasi baru di halaman ini dan memperbarui tanggal "Terakhir diperbarui".

Hubungi Kami
